# insurance-risk-and-claims-modeling
End-to-end data science project focused on insurance risk, claim frequency, and severity modeling using real-world datasets, with emphasis on statistical rigor, interpretability, and business impact.

## Data Sources

This project uses the French Motor Third-Party Liability (MTPL) dataset:

- freMTPL2freq.csv — claim frequency data
- freMTPL2sev.csv — claim severity data

Source:
https://www.openml.org/d/41214

Due to file size constraints, raw data is not stored in the repository.
Users should download the CSV files and place them in:

data/raw/
